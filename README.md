# 我的书签 - 个人书签导航

一个简洁、高效的个人书签管理工具，支持多级分类、快捷键操作、本地搜索和浏览器书签导入。采用响应式设计，可在桌面和移动端流畅使用。

## ✨ 功能特性

- **快捷新增**：在任何页面按下 `Ctrl+Shift+V` 快速添加书签，自动读取剪贴板网址并抓取网页标题、描述和图标。
- **分类管理**：支持无限级分类，可自由创建、编辑、删除分类，并支持优先级排序（数字越小越靠前）。
- **智能搜索**：内置本地搜索（实时过滤卡片）和多搜索引擎（谷歌、百度、必应、GitHub、Bilibili、DeepSeek）快速跳转。
- **书签导入**：一键导入浏览器导出的书签 HTML 文件，自动保留文件夹层级和图标（Base64 格式）。
- **认证保护**：写操作（新增、编辑、删除、导入）需要 HTTP 基本认证，防止他人篡改数据；浏览公开。
- **私密书签**：支持将书签标记为私密，只有登录用户才能查看。
- **推荐系统**：根据书签点击次数自动生成“推荐”分类，展示最常用的 30 个网页。
- **标签系统**：为书签添加标签，支持通过标签快速过滤和搜索。
- **图标本地化**：自动下载网站图标到本地存储，避免重复网络请求，提升加载速度。
- **响应式布局**：侧边栏可折叠，折叠后鼠标悬停显示子分类，移动端自动适配。
- **动画流畅**：分类展开/折叠、侧边栏切换均有平滑过渡效果。

## 🛠️ 技术栈

- **后端**：Flask + Gunicorn
- **前端**：Bootstrap 5 + Font Awesome 6 + 原生 JavaScript
- **数据存储**：JSON 文件（易于备份和迁移）
- **部署**：Docker + GitHub Actions 自动构建

## 🚀 快速开始

### 本地开发

1. 克隆仓库：

```bash
git clone https://github.com/relimc/bookmarks.git
cd bookmarks
```

2. 创建虚拟环境并安装依赖：

```bash
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate
pip install -r requirements.txt
```

3. 运行开发服务器：

```bash
python main.py
```

## Docker 部署（推荐）

1. 确保已安装 Docker 和 Docker Compose。

2. 创建目录并下载 docker-compose.yml：

```bash
mkdir bookmarks && cd bookmarks
curl -O https://raw.githubusercontent.com/relimc/bookmarks/main/docker-compose.yml 
```
3. 根据需要修改环境变量（用户名、密码）和端口映射。

4. 启动服务：

```bash
docker-compose up -d
```

访问 http://服务器IP:5000。

## ⚙️ 配置说明

### 环境变量

| 变量名 | 说明 | 默认值 |
|--------|------|--------|
| `ADMIN_USERNAME` | 管理员用户名 | `admin` |
| `ADMIN_PASSWORD` | 管理员密码 | `change_this_password` |

生产环境**必须修改默认密码**，建议使用强密码。

### 数据持久化

通过 Docker 卷将容器内的 `/app/data` 和 `/app/static/favicons` 目录挂载到宿主机，`data.json` 和下载的图标文件将保存在宿主机，避免容器销毁导致数据丢失。

## 📖 使用指南

### 添加快捷键
- 按下 `Ctrl+Shift+V` 打开新增弹窗，网址自动填充剪贴板内容。
- 也可点击侧边栏底部的快捷键提示区域打开弹窗。

### 分类操作
- **一级分类**：点击分类名称可展开/折叠子分类，同时右侧显示该分类及其所有子分类的网址。
- **折叠侧边栏**：点击头部折叠按钮，侧边栏仅显示图标，鼠标悬停在一级分类图标上可查看子分类列表。
- **新增分类**：在新增/编辑弹窗中点击“新增分类”并输入名称、选择图标、设置优先级和上级分类。
- **修改分类**：通过“分类管理”弹窗可编辑分类名称、图标、优先级和上级分类，支持删除（自动清理子分类和书签）。

### 标签功能
- 为书签添加标签（多个标签用 `/` 分割），卡片下方会显示圆角标签。
- 点击标签可快速搜索所有包含该标签的书签。

### 推荐分类
- 系统自动统计书签的点击次数，在“推荐”分类中展示点击次数最高的 30 个书签。
- 只有登录用户点击书签才会增加点击次数，未登录用户只跳转不计数。

### 私密书签
- 新增或编辑书签时，可勾选“私密”选项。
- 私密书签仅登录用户可见，未登录用户无法查看。

### 搜索功能
- 下拉菜单选择“本地搜索”可在当前页面过滤卡片（标题、描述、网址、标签）。
- 选择其他搜索引擎后，输入关键词按回车或点击搜索按钮，将在新标签页打开对应搜索引擎结果。

### 导入浏览器书签
1. 从浏览器导出书签为 HTML 文件。
2. 点击页面右上角的“导入书签”按钮，选择文件。
3. 确认导入，系统将自动创建分类并添加书签（包含原图标和层级结构）。

## 🔒 安全说明

- 所有写操作（新增、编辑、删除、导入）均受 HTTP 基本认证保护，仅管理员可执行。
- **强烈建议在生产环境启用 HTTPS**，可通过 Nginx 反向代理配置 SSL 证书，避免密码明文传输。
- 数据文件 `data.json` 和图标文件存储在挂载卷中，请确保宿主机目录权限安全。
